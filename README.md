# justmalhar.com

Canonical home for:
- public essays
- technical/operator notes
- email-gated lead magnets
- paid digital products
- future premium memberships

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- MDX (planned)
- Supabase (planned)
- Resend (planned)
- Gumroad / Razorpay links for v1 payments

## Getting started
```bash
npm install
npm run dev
```
