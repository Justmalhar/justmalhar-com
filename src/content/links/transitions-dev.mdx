[Transitions.dev](https://transitions.dev/) by Jakub Antalik is a collection of essential UI transitions for web apps. Each pattern is interactive, so you can see the motion before copying it.

Highlights:
- Covers modals, menus, tabs, tooltips, skeleton loaders, error states, and more
- Available in CSS and React variants
- Includes a skill file for using transitions directly in AI-assisted workflows

A practical reference when you want polished micro-interactions without rebuilding the same transitions from scratch.
