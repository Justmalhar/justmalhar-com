[animations.dev](https://animations.dev/) is Emil Kowalski's course on web animation — the person behind Sonner, Vaul, and much of the motion craft at Linear and Vercel.

What stands out:
- Deep theory on easing, timing, springs, and when not to animate
- Interactive examples so you feel the difference, not just read about it
- Practical modules on CSS animations and Framer Motion, plus walkthroughs like Dynamic Island and drawer patterns
- Includes a SKILL.md file for animation guidance in coding agents

If you want interfaces that feel intentional rather than accidentally animated, this is the best resource I know.
