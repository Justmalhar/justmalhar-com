[CometChat's React UI Kit](https://www.cometchat.com/react-chat-ui-kit/) provides pre-built chat components you can drop into a React app without wiring up every message state, listener, and UI edge case yourself.

Useful when you need:
- Conversations, message lists, composers, and user lists as composable components
- Reactions, threads, read receipts, presence, and audio recording out of the box
- Support for React, Next.js, React Router, and Astro

A solid option when chat is a feature rather than the entire product and you want to ship faster than building from raw SDK primitives.
